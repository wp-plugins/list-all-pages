List all pages on a WordPress site for easy browsing and editing.

== Description ==

This plugin adds a toggle button to the top of the WordPress admin sidebar 
that will list all of the pages in your site. Clicking on the link to the page 
will immediately take you to the edit screen for that page, giving you quick 
access to view or edit any page from anywhere in the WordPress admin.

== Installation ==

1. Upload the `list-all-pages` directory to the `/wp-content/plugins/`
directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
